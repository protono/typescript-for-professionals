# typescript-for-professionals

## setup

`npm init -y`
`npm i typescript`
`npx tsc --init --rootdir src --outdir lib`
`npx tsc --watch`

## running

`npm run build`
`npx serve build`

## setup 2

`npx tsc --init --rootdir src --outdir lib --sourceMap --declaration --declarationMap`
`npm publish`
